# PharmaPlus

Sistema desenvolvido para auxiliar na gestão de uma farmácia, com controle de estoque, pedidos e autenticação de colaboradores.

## Tecnologias utilizadas

- Front end: HTML, CSS, JavaScript
- Back end: Java + Spring Boot
- Banco de Dados: H2
